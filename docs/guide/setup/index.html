Setup

Use from CDN

Using Adapter from content delivery network (CDN) is the easiest way.

html
<script type="module">
    import { Adapter } from 'https://cdn.jsdelivr.net/npm/@devcapsule/adapter/+esm'
</script>

NPM

Install using NPM is recommended for production use.

shell
$ npm install @devcapsule/adapter

It's recommended to use Javascript Building Tools such as ESBuild, Parcel, Rollup etc. Please read thier documentation. After setting up, you can import Adapter in javascript

Javascript
import { Adapter } from '@devcapsule/adapter';
